# Hardware Inventory Android Application

## Build APK
1. Install Android Studio.
2. Extract this ZIP.
3. Open the `HardwareInventoryAndroid` folder.
4. Allow Gradle Sync and install Android SDK API 37 if requested.
5. Select **Build > Build APK(s)**.
6. APK location: `app/build/outputs/apk/debug/app-debug.apk`.

## Features
Offline dashboard, employees, assets, assignments, returns, transfers, history, search, filters, and CSV export to `Downloads/Hardware Inventory`.

## Important
Records are stored on the device. Export backups regularly. Uninstalling the app or clearing app data can remove records. A future cloud version is needed for synchronized use across multiple devices.
